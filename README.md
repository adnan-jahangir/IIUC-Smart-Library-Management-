# IIUC Smart Library Management

A MERN-stack library management application designed for IIUC. This repository contains an Express/Node backend and a Vite + React frontend, optimized for smooth library operations and collaborative development.

## Project Structure

Top-level layout of the repository:
- `backend/` — Node/Express server, API routes, models, controllers, and database configurations.
- `frontend/` — Vite + React + Tailwind CSS client application.
- `.env.example` — Template for environment variables (redacted).
- `README.md` — This file.

## Prerequisites

Make sure you have the following installed on your local machine before starting:
- **Node.js** (v16+ or LTS recommended)
- **npm** (comes with Node)
- **MongoDB** (Local Community Server or MongoDB Atlas Cloud URI)
- **Git**
